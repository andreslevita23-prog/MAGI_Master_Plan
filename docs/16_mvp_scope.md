# 16. MVP Scope

## Incluye

- Monitoreo periodico de `EURUSD` cada 5 minutos.
- Analisis por tres magos con contrato comun.
- Activacion de `entry_case` y `management_case`.
- Decision determinista del CEO.
- Parametrizacion determinista de entrada, `SL` y `TP`.
- Ejecucion basica via `Bot B`.
- Auditoria completa via `Bot C`.

## No incluye

- ML real en produccion.
- Multiples instrumentos.
- Mas de una posicion viva.
- Casos paralelos.
- Trailing avanzado.
- Optimizacion automatica.

## Criterios de exito

- Coherencia de decisiones.
- Respeto de reglas duras.
- Trazabilidad de punta a punta.
- Dataset util para fases siguientes.
