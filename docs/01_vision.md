# 01. Vision

## Mision

MAGI es el nucleo de decision de Prosperity: un sistema de trading modular, explicable y disciplinado que prioriza control de riesgo, trazabilidad y evolucion gradual.

## Objetivos estrategicos

- Validar una arquitectura operable y auditable.
- Tomar decisiones claras sobre `EURUSD` con una sola operacion viva a la vez.
- Generar datos limpios para evolucion futura hacia ML.
- Evitar cajas negras, acoplamiento excesivo y sobregestion.

## Filosofia operativa

- `Bot A` observa y estructura.
- `Melchor`, `Baltasar` y `Gaspar` interpretan.
- `CEO-MAGI` arbitra.
- `Bot B` ejecuta.
- `Bot C` registra.
- `Bot D` optimizara mas adelante.

## Principios de diseno

- Riesgo primero.
- Contratos pequenos y estables.
- Casos activados solo por necesidad real de accion.
- Unicidad operativa para simplificar auditoria.
- Arquitectura lista para integrar ML sin redisenos mayores.
